# Trabajo Práctico Integrador - IFTS18 - Desarrollo de Sistemas Backend

## Clonar repositorio

Usando HTTPS:

- `git clone https://github.com/matias-ac/tp-desarrollo-sistemas-backend.git`

Usando SSH:

- `git clone git@github.com:matias-ac/tp-desarrollo-sistemas-backend.git`

## Crear entorno virtual local

Ejecutar el comando: 

- `python -m venv venv`

## Activa el entorno virtual

En Windows:

- `.\venv\Scripts\activate`

En MacOS/Linux:

- `source venv/bin/activate`

## Instalar las dependencias

Ejecutar el comando:

- `pip install -r requirements.txt`

## Crear variables de entorno

1. Crear un archivo `.env`
2. Copiar y pegar el contenido del archivo de ejemplo `.env.example`
3. Actualizar la variable `SECRET_KEY`
